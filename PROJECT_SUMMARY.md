## 🎮 功能演示

### 支持的诈骗类型（5种）

1. **🎭 虚假服务** - 伪装成正规服务机构的诈骗
2. **👤 身份冒充** - 冒充熟人、官方机构进行诈骗
3. **🎣 钓鱼诈骗** - 通过虚假链接窃取个人信息
4. **💼 虚假招聘** - 以高薪招聘为名骗取钱财
5. **💕 网恋诈骗** - 通过感情关系骗取钱财

### 互动学习流程

```
选择场景 → 接收模拟消息 → 用户判断 → AI分析 → 透明度展示 → 学习反馈
```

### 示例场景分析

**场景**：虚假驾校服务
```
消息："您好，这里是环城驾校，我们可以代办B1驾照，
      无需考试，只需支付800元定金和7500元考试费..."

风险点：
- ⚠️ 承诺无需考试
- ⚠️ 要求预付费用  
- ⚠️ 个人账户收款

用户响应分析：
- 安全得分：85/100
- 识别的风险：要求预付费用
- 遗漏的风险：承诺无需考试、个人账户收款
```
